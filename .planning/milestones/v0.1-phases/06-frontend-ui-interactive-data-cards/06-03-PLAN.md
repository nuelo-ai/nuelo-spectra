  <action>
    1. Create `frontend/src/stores/tabStore.ts`:
       - Define `FileTab` interface: `{ fileId: string; fileName: string }`
       - Define `TabStore` interface with state and actions:
         - `tabs: FileTab[]` -- currently open tabs
         - `currentTabId: string | null` -- active tab file ID
         - `maxTabs: number` (constant 5)
         - `openTab(fileId: string, fileName: string): boolean` -- opens tab, returns false if at max
         - `closeTab(fileId: string): void` -- closes tab, auto-switches to next available
         - `switchTab(fileId: string): void` -- sets currentTabId
         - `closeTabForDeletedFile(fileId: string): void` -- removes tab for deleted file, switches to next
       - `openTab` logic:
         - If file already has open tab: switch to it, return true
         - If tabs.length >= maxTabs: return false (caller shows alert)
         - Otherwise: add tab, set as current, return true
       - `closeTab` logic:
         - Remove tab from list
         - If closed tab was current: switch to first remaining tab or null
       - Export `useTabStore` hook via `create<TabStore>()`

    2. Create `frontend/src/hooks/useFileManager.ts`:
       - `useFiles()`: TanStack Query hook wrapping GET /api/files/
         - queryKey: ['files']
         - Returns `FileListItem[]`
         - Refetch on window focus
       - `useUploadFile()`: TanStack Query mutation wrapping POST /api/files/upload
         - Accepts `{ file: File, userContext?: string }`
         - Creates FormData, appends file and user_context
         - Uses apiClient.upload()
         - On success: invalidate ['files'] query
         - Returns `FileUploadResponse`
       - `useDeleteFile()`: TanStack Query mutation wrapping DELETE /api/files/{fileId}
         - On success: invalidate ['files'] query
       - `useFileSummary(fileId: string)`: TanStack Query hook wrapping GET /api/files/{fileId}/summary
         - queryKey: ['files', 'summary', fileId]
         - Returns `FileSummaryResponse`
         - enabled: only when fileId is provided
         - Polls every 3 seconds until data_summary is not null (for onboarding completion)
  </action>

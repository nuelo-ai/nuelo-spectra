  <action>
    1. Create `frontend/src/components/data/DownloadButtons.tsx` (Client Component):
       - Two variants: CSV download and Markdown download

       **CSVDownloadButton:**
       - Props: `data: { columns: string[], rows: Record<string, any>[] }`, `filename?: string`
       - Uses `papaparse` library's `Papa.unparse()` to generate CSV string
       - Creates Blob with MIME type `text/csv;charset=utf-8;`
       - Uses `URL.createObjectURL()` and programmatic link click to trigger download
       - Cleans up URL after download with `URL.revokeObjectURL()`
       - Default filename: `spectra-export-{timestamp}.csv`
       - Button: shadcn Button variant="outline" size="sm" with Lucide `Download` icon
       - Button text: "Download CSV"
       - Disabled if no data

       **MarkdownDownloadButton:**
       - Props: `queryBrief: string`, `explanation: string`, `tableData?: { columns: string[], rows: Record<string, any>[] }`, `filename?: string`
       - Generates Markdown string:
         ```markdown
         # {queryBrief}

         ## Analysis

         {explanation}

         ## Data

         | col1 | col2 | col3 |
         |------|------|------|
         | val1 | val2 | val3 |
         ```
       - Creates Blob with MIME type `text/markdown;charset=utf-8;`
       - Same download pattern as CSV
       - Default filename: `spectra-analysis-{timestamp}.md`
       - Button: shadcn Button variant="outline" size="sm" with Lucide `FileText` icon
       - Button text: "Download Markdown"
       - If tableData present: include as Markdown table in export

    2. Create `frontend/src/components/data/CodeDisplay.tsx` (Client Component):
       - Props: `code: string`, `language?: string` (default "python")
       - Renders code in a styled pre/code block
       - Features:
         - Copy-to-clipboard button (Lucide `Copy` icon -> `Check` icon for 2s after copy)
         - Uses `navigator.clipboard.writeText(code)` for copy
         - Language label badge (e.g., "Python") in top-right corner
         - Monospace font (font-mono)
         - Dark background (slate-900) with light text for code contrast
         - Rounded corners, padding
         - Horizontal scroll for long lines (overflow-x-auto, whitespace-pre)
         - Line numbers optional (show if code has > 3 lines)
       - Do NOT use a heavy syntax highlighting library (keeps bundle small)
       - Simple styling: dark background, monospace, subtle line numbering
       - Collapsible: starts collapsed with "View code" toggle button
         - Toggle text: "View code" / "Hide code"
         - Uses Lucide `Code` icon next to toggle text
         - Smooth expand/collapse animation
  </action>

    Create backend/app/agents/llm_factory.py with factory function for multi-provider LLM support:

    1. **get_llm(provider: str, model: str, api_key: str, **kwargs) -> BaseChatModel:**
       - Takes provider name ("anthropic", "openai", "google")
       - Takes model name (e.g., "claude-sonnet-4-20250514", "gpt-4", "gemini-pro")
       - Takes API key for the selected provider
       - Additional kwargs passed through (max_tokens, temperature, etc.)
       - Returns LangChain BaseChatModel instance

       Implementation:
       ```python
       from langchain_core.language_models import BaseChatModel
       from typing import Any

       def get_llm(
           provider: str,
           model: str,
           api_key: str,
           **kwargs: Any
       ) -> BaseChatModel:
           """Create LLM instance based on provider.

           Args:
               provider: LLM provider name (anthropic, openai, google)
               model: Model name/ID
               api_key: API key for the provider
               **kwargs: Additional model parameters (max_tokens, temperature, etc.)

           Returns:
               LangChain BaseChatModel instance

           Raises:
               ValueError: If provider is not supported
           """
           provider = provider.lower()

           if provider == "anthropic":
               from langchain_anthropic import ChatAnthropic
               return ChatAnthropic(model=model, api_key=api_key, **kwargs)

           elif provider == "openai":
               from langchain_openai import ChatOpenAI
               return ChatOpenAI(model=model, api_key=api_key, **kwargs)

           elif provider == "google":
               from langchain_google_genai import ChatGoogleGenerativeAI
               return ChatGoogleGenerativeAI(model=model, google_api_key=api_key, **kwargs)

           else:
               raise ValueError(
                   f"Unsupported LLM provider: {provider}. "
                   f"Supported providers: anthropic, openai, google"
               )
       ```

    2. Import BaseChatModel from langchain_core.language_models for type hints.

    3. Use lazy imports (import inside if blocks) to avoid requiring all provider SDKs if user only uses one provider.

    4. Design considerations:
       - Factory returns BaseChatModel interface, making all agents provider-agnostic
       - Agent code calls get_llm(settings.llm_provider, settings.agent_model, settings.{provider}_api_key)
       - Easy to add new providers (Cohere, Mistral, local Ollama) by adding elif branches
       - kwargs enables per-agent customization (different max_tokens per agent)

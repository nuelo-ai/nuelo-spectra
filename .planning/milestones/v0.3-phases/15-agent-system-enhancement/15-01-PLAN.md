<task type="auto">
  <name>Task 2: Create ContextAssembler service with progressive reduction and join hints</name>
  <files>backend/app/services/context_assembler.py</files>
  <action>
    Create `backend/app/services/context_assembler.py` as a standalone service class.

    **Class: ContextAssembler**

    **Constructor `__init__(self)`:**
    - Load settings.yaml to get token_budget and safety_margin
    - Create tiktoken encoder using `tiktoken.encoding_for_model("gpt-4")` (same pattern as `TiktokenCounter` in `backend/app/agents/memory/token_counter.py`)
    - Store effective_budget = token_budget * (1 - safety_margin)

    **Method: `async def assemble(self, db: AsyncSession, file_ids: list[UUID], user_id: UUID) -> dict`:**
    - Load file records using `FileService.get_user_file(db, file_id, user_id)` for each file_id
    - Validate ALL files exist and have data_summary (onboarding complete). If any file is missing or not onboarded, raise ValueError with clear message naming the problematic file (per user decision: fail whole query if any file has error)
    - Profile each file using `OnboardingAgent().profile_data(file.file_path, file.file_type)` -- reuse existing profiling logic
    - Build file metadata list with: id (str), name (original_filename), var_name (sanitized), profile (from profiling), size_bytes (file_size)
    - Call `_detect_join_hints(profiles)` to find shared columns
    - Run progressive reduction loop through levels: ["full", "no_samples", "no_stats", "minimal"]
    - For each level, call `_build_context_string(profiles, join_hints, level)` then count tokens
    - Return first level that fits within effective_budget
    - Return dict: `{"files": [...], "join_hints": [...], "context_string": str, "total_tokens": int, "reduction_level": str}`

    **Method: `def _sanitize_var_name(self, filename: str) -> str`:**
    - Use `Path(filename).stem` to remove extension
    - `re.sub(r'[^a-z0-9_]', '_', stem.lower())` to sanitize
    - Strip leading/trailing underscores
    - Prepend `data_` if starts with digit
    - Add `df_` prefix
    - Track used names in a set passed as parameter or instance variable during assemble() call
    - On collision, append `_2`, `_3` etc. (per Pitfall 2 in research)

    **Method: `def _detect_join_hints(self, profiles: list[dict]) -> list[str]`:**
    - For each pair of files (i, j where i < j), find shared column names via set intersection
    - For each shared column, build hint string: `"Possible join: {file_a_name}.{col} ({dtype_a}) <-> {file_b_name}.{col} ({dtype_b})"`
    - If dtype_a != dtype_b, append ` [WARNING: Type mismatch]` (per Pitfall 3 in research)
    - Return list of hint strings

    **Method: `def _build_context_string(self, profiles: list[dict], join_hints: list[str], level: str) -> str`:**
    - Build markdown-formatted context string:
      - Header: `# Multi-File Dataset Context`
      - For each file: `## File: {name}`, `Variable name: {var_name}`, rows/columns count
      - Column list with dtype always included
      - `level="full"`: include dtype + stats (mean, min, max) + sample data (first 3 rows)
      - `level="no_samples"`: include dtype + stats, omit sample data
      - `level="no_stats"`: include dtype only, no stats, no samples
      - `level="minimal"`: column names + types only, one line per column
    - Append join hints section if any exist
    - Return joined string

    **Method: `def _count_tokens(self, text: str) -> int`:**
    - Use `len(self.encoder.encode(text))`

    **Helper: `def load_session_settings() -> dict`:**
    - Module-level function that loads settings.yaml (following same pattern as `load_prompts()` in `backend/app/agents/config.py`)
    - Cache with `@lru_cache` or similar
    - Return parsed YAML dict

    Key implementation notes:
    - Use `from __future__ import annotations` for forward references
    - Import OnboardingAgent inside methods to avoid circular imports (same pattern as agent_service.py line 398)
    - The _sanitize_var_name collision tracking should be scoped per assemble() call (pass used_names set)

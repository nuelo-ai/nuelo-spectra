     - Cell: File type icon (FileSpreadsheet for csv/xlsx/xls, FileText for others from lucide-react) + filename text

     - Right-aligned, muted text

     - Cell: formatted date (e.g., "Feb 11, 2026") using `new Date(value).toLocaleDateString("en-US", { month: "short", day: "numeric", year: "numeric" })`
     - Muted text

     - Cell: MessageSquarePlus icon button (per user decision: "Per-row inline icon: chat icon (start new chat with this file)")

       - "View context" (Eye icon) — opens FileContextModal (set state: contextFileId)
       - "Download" (Download icon) — calls useDownloadFile mutation
       - Separator
       - "Delete" (Trash2 icon, text-destructive) — opens delete AlertDialog

  - Shows file count: "{files.length} files" when nothing selected

  - Delete Selected opens AlertDialog with: "Delete {selectedCount} file(s)? This cannot be undone."
  - On confirm: call useBulkDeleteFiles with selected file IDs, then clear rowSelection, toast.success("{count} file(s) deleted")

  **Per-row delete confirmation:**
  - Single file delete uses AlertDialog: "Delete file? This cannot be undone." (per user decision)

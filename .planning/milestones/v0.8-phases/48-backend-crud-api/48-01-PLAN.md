```python
async def require_workspace_access(
    current_user: CurrentUser,
) -> User:
    """Verify user's tier has workspace_access=True. Raises 403 if not."""
    config = get_class_config(current_user.user_class)
    if not config or not config.get("workspace_access", False):
        raise HTTPException(
            status_code=status.HTTP_403_FORBIDDEN,
            detail="workspace access not available on your plan"
        )
    return current_user

WorkspaceUser = Annotated[User, Depends(require_workspace_access)]
```

Methods needed:

**Collection CRUD:**
- `create_collection(db, user_id, name, description=None) -> Collection` -- create and return
- `list_user_collections(db, user_id) -> list[dict]` -- SELECT Collection + COUNT subqueries for file_count, signal_count; return list of dicts with keys "collection", "file_count", "signal_count"; order by created_at DESC
- `get_user_collection(db, collection_id, user_id) -> Collection | None` -- single collection filtered by user_id
- `get_collection_detail(db, collection_id, user_id) -> dict | None` -- Collection + file_count, signal_count, report_count via COUNT subqueries; return dict or None
- `update_collection(db, collection_id, user_id, name=None, description=None) -> Collection | None` -- partial update, return updated or None if not found
- `count_user_collections(db, user_id) -> int` -- COUNT for limit check

**Collection Files:**
- `add_file_to_collection(db, collection_id, file_id) -> CollectionFile` -- create junction row
- `get_collection_file(db, collection_id, file_id) -> CollectionFile | None` -- check existence (for duplicate prevention)
- `list_collection_files(db, collection_id, user_id) -> list[CollectionFile]` -- files in collection with eagerly loaded File relationship (for filename, file_size, data_summary); verify collection ownership
- `remove_file_from_collection(db, collection_id, file_id, user_id) -> bool` -- delete CollectionFile row (NOT the File itself); verify collection ownership; return True if deleted, False if not found

**Reports:**
- `list_collection_reports(db, collection_id, user_id) -> list[Report]` -- reports for collection, verify ownership, order by created_at DESC
- `get_report(db, report_id, collection_id, user_id) -> dict | None` -- report detail with signal_count (COUNT signals WHERE pulse_run_id = report.pulse_run_id, handling None pulse_run_id gracefully with COALESCE)

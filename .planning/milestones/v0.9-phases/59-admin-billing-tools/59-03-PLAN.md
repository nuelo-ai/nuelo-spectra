**1. Create useBilling.ts** (`admin-frontend/src/hooks/useBilling.ts`):
Follow the exact hook pattern from useUsers.ts. Use `adminApiClient` for all calls.

```typescript
"use client";
import { useMutation, useQuery, useQueryClient } from "@tanstack/react-query";
import { adminApiClient } from "@/lib/admin-api-client";
```

Hooks to create:
- `useUserBillingDetail(userId: string | undefined)` — `useQuery` fetching `GET /api/admin/billing/users/${userId}`. queryKey: `["admin", "billing", userId]`. enabled: `!!userId`.
- `useForceSetTier()` — `useMutation` posting to `/api/admin/billing/users/${userId}/force-set-tier` with body `{ new_tier, reason }`. On success, invalidate `["admin", "billing"]` and `["admin", "users"]`.
- `useAdminCancelSubscription()` — `useMutation` posting to `/api/admin/billing/users/${userId}/cancel`. Invalidate same keys.
- `useRefundPayment()` — `useMutation` posting to `/api/admin/billing/users/${userId}/refund` with body `{ payment_id, amount_cents }`. Invalidate same keys.
- `useBillingSettings()` — `useQuery` fetching `GET /api/admin/billing-settings`. queryKey: `["admin", "billing-settings"]`.
- `useUpdateBillingSettings()` — `useMutation` PUT `/api/admin/billing-settings`. Invalidate `["admin", "billing-settings"]`.

Each mutation follows the `useUserMutation` pattern: check `res.ok`, throw `new Error(body.detail || "Operation failed")`.

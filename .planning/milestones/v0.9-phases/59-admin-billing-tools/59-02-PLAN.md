**2. Create discount code service** (`backend/app/services/discount_code.py`):
```python
class DiscountCodeService:
    """Handles discount code CRUD with Stripe Coupon/Promotion Code sync."""
```

Static async methods:

`create(db, code, discount_type, discount_value, max_redemptions, expires_at, admin_id)`:
- Auto-uppercase the code string
- Create Stripe Coupon via `client.v1.coupons.create()`:
  - If percent_off: `params={"percent_off": discount_value, "duration": "forever", "metadata": {"source": "spectra_admin", "code": code}}`
  - If amount_off: `params={"amount_off": discount_value, "currency": "usd", "duration": "once", "metadata": {"source": "spectra_admin", "code": code}}`
  - Add `"max_redemptions": max_redemptions` if not None
  - Add `"redeem_by": int(expires_at.timestamp())` if expires_at is not None
- Create Stripe Promotion Code via `client.v1.promotion_codes.create()`:
  `params={"coupon": coupon.id, "code": code, "active": True}`
  - Add `"max_redemptions": max_redemptions` if not None
  - Add `"expires_at": int(expires_at.timestamp())` if expires_at is not None
- Per D-21: BOTH Coupon and Promotion Code must be created.
- Save DiscountCode row with stripe_coupon_id=coupon.id, stripe_promotion_code_id=promo.id
- Return the created row

`list_all(db)`:
- Query all DiscountCode rows ordered by created_at desc
- Return list + count

`update(db, discount_code_id, max_redemptions, expires_at)`:
- Update local fields only (Stripe coupons are largely immutable for amounts)
- If promotion code exists and expires_at changed, note: Stripe promotion codes cannot update expires_at after creation. Log warning but update locally.

`deactivate(db, discount_code_id)`:
- Per D-18: Set is_active=False locally
- Per ADMIN-09: Deactivate Stripe Promotion Code: `client.v1.promotion_codes.update(promo_id, params={"active": False})`

`delete(db, discount_code_id)`:
- Deactivate Stripe Promotion Code first (if exists)
- Delete Stripe Coupon: `client.v1.coupons.delete(coupon_id)`
- Delete local row from DB

Use `SubscriptionService._get_stripe_client()` to get the Stripe client (reuse pattern).

**3. Create discount codes router** (`backend/app/routers/admin/discount_codes.py`):
```python
router = APIRouter(prefix="/discount-codes", tags=["admin-discount-codes"])
```

Endpoints:
- `GET /discount-codes` -- List all discount codes. Returns DiscountCodeListResponse.
- `POST /discount-codes` -- Create discount code. Accepts CreateDiscountCodeRequest. Calls DiscountCodeService.create(). Log admin action. `await db.commit()`.
- `PUT /discount-codes/{code_id}` -- Update discount code. Accepts UpdateDiscountCodeRequest. Calls DiscountCodeService.update(). `await db.commit()`.
- `POST /discount-codes/{code_id}/deactivate` -- Deactivate. Calls DiscountCodeService.deactivate(). Log admin action. `await db.commit()`.
- `DELETE /discount-codes/{code_id}` -- Delete. Calls DiscountCodeService.delete(). Log admin action. `await db.commit()`.

Every endpoint uses `CurrentAdmin` and `DbSession` dependencies.

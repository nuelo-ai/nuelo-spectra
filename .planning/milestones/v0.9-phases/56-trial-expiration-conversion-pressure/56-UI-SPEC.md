## Interaction States

### Banner State Machine

```
[Not Trial User] -> (hidden)
[Trial User, dismissed=true] -> (hidden)
[Trial User, days>3 AND credits>10] -> Normal Banner (blue, Clock icon, outline CTA)
[Trial User, days<=3 OR credits<=10] -> Urgent Banner (amber, AlertTriangle icon, filled CTA)
[Trial User, expired] -> (hidden â€” overlay takes over)
```

### Overlay State Machine

```
[Not Trial User] -> (hidden)
[Trial User, not expired] -> (hidden)
[Trial User, expired, on /settings/*] -> (hidden â€” settings accessible)
[Trial User, expired, on dashboard/workspace] -> Visible Overlay
```

### 402 Response Handling

```
[API returns 402] -> fetchWithAuth emits "trial-expired" CustomEvent
                  -> Does NOT attempt token refresh (checked before 401 handler)
                  -> Overlay renders based on user state from /auth/me (primary trigger)
                  -> 402 is server-side safety net, not primary UI trigger
```

  <files>backend/app/routers/admin/credits.py, backend/app/routers/credits.py</files>
  <action>
Create `backend/app/routers/admin/credits.py` with `router = APIRouter(prefix="/credits", tags=["admin-credits"])`. Use CurrentAdmin and DbSession from dependencies.py. Import CreditService from services.credit, log_admin_action from services.admin.audit, verify_password from utils.security.

Endpoints:

**GET /users/{user_id}** - View user's credit balance (CREDIT-06):
- Query User by user_id (404 if not found).
- Call `CreditService.get_balance(db, user_id, user.user_class)`.
- Return CreditBalanceResponse.

**GET /users/{user_id}/transactions** - View transaction history (CREDIT-08):
- Accept query params: limit (int, default=50, max=200), offset (int, default=0).
- Call `CreditService.get_transaction_history(db, user_id, limit, offset)`.
- Return list of CreditTransactionResponse.
- Audit log: action="view_credit_history", target_type="user_credit", target_id=str(user_id).

**POST /users/{user_id}/adjust** - Adjust credits (CREDIT-07):
- Accept CreditAdjustmentRequest body (amount, reason, password).
- Verify admin password: `verify_password(body.password, current_admin.hashed_password)`. If fails, raise HTTPException 401 "Invalid password" (per locked decision: admin must re-enter password).
- Call `CreditService.admin_adjust(db, user_id, body.amount, body.reason, current_admin.id)`.
- Audit log: action="credit_adjustment", target_type="user_credit", target_id=str(user_id), details={"amount": float(body.amount), "reason": body.reason, "balance_after": float(result.balance)}.
- `await db.commit()`.
- Return CreditBalanceResponse.

**POST /users/{user_id}/reset** - Manual credit reset (CREDIT-12, CREDIT-13):
- Accept CreditManualResetRequest body (password).
- Verify admin password (same pattern as adjust).
- Call `CreditService.manual_reset(db, user_id, current_admin.id)`.
- Audit log: action="credit_manual_reset", target_type="user_credit", target_id=str(user_id).
- `await db.commit()`.
- Return CreditBalanceResponse.

**GET /distribution** - Credit distribution by class (CREDIT-09 backend):
- Call `CreditService.get_credit_distribution(db)`.
- Return list of dicts with user_class, count, avg_balance.

**GET /low-balance** - Low credit users list (CREDIT-10 backend):
- Accept query param: threshold_pct (float, default=0.1).
- Call `CreditService.get_low_credit_users(db, threshold_pct)`.
- Return list of dicts with user info and balance details.

For all admin endpoints, use `request: Request` parameter to get IP via `request.client.host if request.client else None` for audit logging.

Create `backend/app/routers/credits.py` with `router = APIRouter(prefix="/api/credits", tags=["credits"])`. Use CurrentUser and DbSession from dependencies.py.

**GET /balance** - Public user's own credit balance:
- Call `CreditService.get_balance(db, current_user.id, current_user.user_class)`.
- Return CreditBalanceResponse.
- This is the endpoint the frontend sidebar will call (Phase 31).

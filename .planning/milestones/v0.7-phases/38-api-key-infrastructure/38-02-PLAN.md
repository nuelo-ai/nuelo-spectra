  <behavior>
Key generation contract:
- `generate_api_key()` → `(full_key: str, key_prefix: str, token_hash: str)`
- full_key format: `spe_<43-char-base64url>` (total ~47 chars)
- key_prefix: `full_key[:12]` (e.g. "spe_abc123de")
- token_hash: `hashlib.sha256(full_key.encode()).hexdigest()` (64 hex chars)

Cases: `generate_api_key()` → full_key starts with "spe_", len(token_hash)==64, key_prefix==full_key[:12]

Service authentication contract:
- `ApiKeyService.authenticate(db, valid_active_key)` → User object
- `ApiKeyService.authenticate(db, revoked_key)` → None
- `ApiKeyService.authenticate(db, "invalid-key")` → None
- `ApiKeyService.authenticate(db, "spe_" + "x"*43)` → None (hash not in DB)

Revocation contract:
- `ApiKeyService.revoke(db, key_id, owner_user_id)` → True
- `ApiKeyService.revoke(db, key_id, wrong_user_id)` → False (key not found for this user)
- `ApiKeyService.revoke(db, nonexistent_key_id, any_user_id)` → False

Test approach: Use unittest.mock to mock AsyncSession (db) rather than hitting a real DB. Mock `db.execute()` return values to simulate found/not-found API key records. Use `AsyncMock` for async session methods. This follows the test_routing.py pattern of mocking dependencies rather than running a live server.
  </behavior>

**Schemas first** (`backend/app/schemas/api_key.py`):

```python
from pydantic import BaseModel, Field
from datetime import datetime
from uuid import UUID

class ApiKeyCreateRequest(BaseModel):
    name: str = Field(..., min_length=1, max_length=100)
    description: str | None = Field(None, max_length=500)

class ApiKeyCreateResponse(BaseModel):
    id: UUID
    name: str
    key_prefix: str
    full_key: str   # ONLY returned from POST — never from GET
    created_at: datetime

class ApiKeyListItem(BaseModel):
    id: UUID
    name: str
    description: str | None
    key_prefix: str
    is_active: bool
    created_at: datetime
    last_used_at: datetime | None

    model_config = {"from_attributes": True}
```

**Service** (`backend/app/services/api_key.py`):

Static method class following CreditService pattern:
- `generate_api_key() -> tuple[str, str, str]` — module-level function (not static method), used by `create()`
- `create(db, user_id, name, description) -> tuple[ApiKey, str]` — creates record, returns (record, full_raw_key). Uses `db.add()`, `await db.flush()`, returns before commit (caller's endpoint commits via response lifecycle)
- `list_for_user(db, user_id) -> list[ApiKey]` — SELECT WHERE user_id=user_id ORDER BY created_at DESC (no is_active filter — return all including revoked so user can see history)
- `revoke(db, key_id, user_id) -> bool` — SELECT WHERE id=key_id AND user_id=user_id, set is_active=False, flush, return True. If not found return False.
- `authenticate(db, raw_key) -> User | None` — hash raw_key, SELECT WHERE token_hash=hash AND is_active=True, if found update last_used_at via flush, return User via get_user_by_id. Return None on any miss.

Use `from app.services.auth import get_user_by_id` for the user lookup in authenticate().
Import `from sqlalchemy import select` and `from sqlalchemy.ext.asyncio import AsyncSession`.

```python
"""TDD tests for ApiKeyService and generate_api_key()."""
import hashlib
from unittest.mock import AsyncMock, MagicMock, patch
from uuid import uuid4

import pytest

from app.services.api_key import ApiKeyService, generate_api_key


class TestGenerateApiKey:
    def test_full_key_starts_with_spe_prefix(self):
        full_key, _, _ = generate_api_key()
        assert full_key.startswith("spe_")

    def test_token_hash_is_sha256_of_full_key(self):
        full_key, _, token_hash = generate_api_key()
        expected = hashlib.sha256(full_key.encode()).hexdigest()
        assert token_hash == expected

    def test_token_hash_is_64_hex_chars(self):
        _, _, token_hash = generate_api_key()
        assert len(token_hash) == 64
        assert all(c in "0123456789abcdef" for c in token_hash)

    def test_key_prefix_is_first_12_chars_of_full_key(self):
        full_key, key_prefix, _ = generate_api_key()
        assert key_prefix == full_key[:12]

    def test_keys_are_unique(self):
        keys = {generate_api_key()[0] for _ in range(10)}
        assert len(keys) == 10


class TestApiKeyServiceAuthenticate:
    @pytest.mark.asyncio
    async def test_authenticate_valid_active_key_returns_user(self):
        mock_db = AsyncMock()
        mock_api_key = MagicMock()
        mock_api_key.user_id = uuid4()
        mock_api_key.is_active = True
        mock_result = MagicMock()
        mock_result.scalar_one_or_none.return_value = mock_api_key
        mock_db.execute.return_value = mock_result
        mock_user = MagicMock()

        with patch("app.services.api_key.get_user_by_id", return_value=mock_user):
            result = await ApiKeyService.authenticate(mock_db, "spe_validkey")

        assert result is mock_user

    @pytest.mark.asyncio
    async def test_authenticate_invalid_key_returns_none(self):
        mock_db = AsyncMock()
        mock_result = MagicMock()
        mock_result.scalar_one_or_none.return_value = None
        mock_db.execute.return_value = mock_result

        result = await ApiKeyService.authenticate(mock_db, "spe_invalidkey")
        assert result is None

    @pytest.mark.asyncio
    async def test_authenticate_updates_last_used_at(self):
        mock_db = AsyncMock()
        mock_api_key = MagicMock()
        mock_api_key.user_id = uuid4()
        mock_result = MagicMock()

        assert mock_api_key.last_used_at is not None
        mock_db.flush.assert_awaited_once()


class TestApiKeyServiceRevoke:

    async def test_revoke_owned_key_returns_true(self):
        mock_db = AsyncMock()
        user_id = uuid4()
```

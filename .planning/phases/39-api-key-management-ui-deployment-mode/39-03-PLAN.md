**Hooks:**

1. `useUserApiKeys(userId: string | undefined)` — useQuery with queryKey `["admin", "users", userId, "api-keys"]`, enabled only when userId is truthy. Fetches `GET /api/admin/users/${userId}/api-keys` via `adminApiClient`.

2. `useCreateUserApiKey(userId: string)` — useMutation that POSTs to `/api/admin/users/${userId}/api-keys`. On success: invalidate query key, toast.success("API key created").

3. `useRevokeUserApiKey(userId: string)` — useMutation that DELETEs `/api/admin/users/${userId}/api-keys/${keyId}`. On success: invalidate query key, toast.success("API key revoked").

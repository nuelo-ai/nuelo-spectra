  <action>
    1. Create `frontend/src/lib/api-client.ts`:
       - Create a `apiClient` object with methods: `get(path)`, `post(path, body)`, `patch(path, body)`, `delete(path)`, `upload(path, formData)`
       - Each method calls `fetch('/api' + path, ...)` (proxied through Next.js rewrites)
       - Automatically injects `Authorization: Bearer {access_token}` header from localStorage
       - On 401 response: attempt token refresh using stored refresh_token via POST /api/auth/refresh
       - If refresh succeeds: retry original request with new access_token
       - If refresh fails: clear tokens from localStorage, redirect to /login
       - Store tokens in localStorage as `spectra_access_token` and `spectra_refresh_token`
       - Export helper functions: `setTokens(access, refresh)`, `getAccessToken()`, `clearTokens()`, `isAuthenticated()`

    2. Create `frontend/src/hooks/useAuth.ts` (Client Component hook):
       - Manages auth state: `user: UserResponse | null`, `isLoading: boolean`, `isAuthenticated: boolean`
       - On mount: check localStorage for tokens, if found call GET /api/auth/me to validate and load user
       - `login(email, password)`: POST /api/auth/login, store tokens, load user, redirect to /dashboard
       - `signup(email, password, firstName?, lastName?)`: POST /api/auth/signup, store tokens, load user, redirect to /dashboard
       - `logout()`: clear tokens from localStorage, set user to null, redirect to /login
       - `forgotPassword(email)`: POST /api/auth/forgot-password
       - `resetPassword(token, newPassword)`: POST /api/auth/reset-password
       - Use React Context (createContext + Provider) to share auth state across app
       - Export `AuthProvider` component and `useAuth()` hook

    3. Update `frontend/src/app/providers.tsx` to wrap with `AuthProvider` inside QueryClientProvider.

    4. Create `frontend/src/app/(auth)/layout.tsx`:

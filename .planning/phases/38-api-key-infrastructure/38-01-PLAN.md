Create `backend/app/models/api_key.py` as a new file. Model follows the PasswordResetToken/Invitation pattern from the codebase.

The model MUST include:
- `id: Mapped[UUID]` — primary key, default uuid4
- `user_id: Mapped[UUID]` — ForeignKey("users.id", ondelete="CASCADE"), index=True
- `name: Mapped[str]` — String(100), not nullable
- `description: Mapped[str | None]` — Text, nullable=True
- `key_prefix: Mapped[str]` — String(16), not nullable (stores first 12 chars of raw key for display)
- `token_hash: Mapped[str]` — String(128), unique=True, index=True
- `is_active: Mapped[bool]` — Boolean, default=True
- `scopes: Mapped[list[str] | None]` — PG_ARRAY(String), nullable=True (forward compat; NULL = all scopes)
- `expires_at: Mapped[datetime | None]` — DateTime(timezone=True), nullable=True (NULL = no expiry)
- `last_used_at: Mapped[datetime | None]` — DateTime(timezone=True), nullable=True
- `created_at: Mapped[datetime]` — DateTime(timezone=True), default=lambda: datetime.now(timezone.utc)
- `user: Mapped["User"]` relationship — back_populates="api_keys"

Import pattern: use `from sqlalchemy.dialects.postgresql import ARRAY as PG_ARRAY` for scopes column.
Use `TYPE_CHECKING` guard for `from app.models.user import User` import.

- Add `from app.models.api_key import ApiKey` inside `TYPE_CHECKING` block (do NOT add to runtime imports — circular import risk)
- Add relationship to User class after existing relationships:
  ```python
  api_keys: Mapped[list["ApiKey"]] = relationship(
      "ApiKey",
      back_populates="user",
      cascade="all, delete-orphan"
  )
  ```

Do NOT use Argon2 hashing anywhere in this model. SHA-256 is used at service layer (Plan 02). This model is pure storage.

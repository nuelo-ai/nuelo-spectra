# Pydantic v2 schemas with from_attributes=True

class UserSummary(BaseModel):
    """User summary for list view."""
    model_config = ConfigDict(from_attributes=True)
    id: UUID
    email: str
    first_name: str | None
    last_name: str | None

    """Full user detail with aggregated counts."""
    model_config = ConfigDict(from_attributes=True)
    id: UUID
    email: str
    first_name: str | None
    last_name: str | None
    is_active: bool
    is_admin: bool

class UserActivityResponse(BaseModel):
    """Activity timeline for a user."""

**5. Create service** (`backend/app/services/admin/users.py`):

Implement `AdminUserService` class (or module-level functions, following existing pattern in `services/credit.py`):

- `list_users(db, page, per_page, search, is_active, user_class, signup_after, signup_before, sort_by, sort_order)`:
  - Base query: `select(User)` with LEFT JOIN on `UserCredit` for credit_balance.
  - Search: If `search` provided, filter with `or_(User.email.ilike(f"%{search}%"), User.first_name.ilike(f"%{search}%"), User.last_name.ilike(f"%{search}%"))`.
  - Filters: chain `.where()` for each non-None filter. `signup_after`/`signup_before` use `User.created_at >= / <=`.
  - Count total using `select(func.count()).select_from(query.subquery())`.
  - Sort: Map `sort_by` to column (`created_at`, `last_login_at`, `first_name`, `credit_balance`). Apply `asc()` or `desc()` based on `sort_order`. For `credit_balance` sort, use the joined `UserCredit.balance` column.
  - Pagination: `.offset((page - 1) * per_page).limit(per_page)`.
  - Return dict matching `UserListResponse` structure. Build `UserSummary` objects from query results, pulling `credit_balance` from the join.
  - Per page is fixed at 20 (per locked decision), but accept as parameter for the service, enforce in router.

- `get_user_detail(db, user_id)`:
  - Fetch user by ID. Raise ValueError if not found.
  - Run aggregate count queries: `func.count(File.id)`, `func.count(ChatSession.id)`, `func.count(ChatMessage.id)` filtered by `user_id`.
  - Get credit balance from `UserCredit`.
  - Compute `last_message_at` via `select(func.max(ChatMessage.created_at)).where(ChatMessage.user_id == user_id)`.
  - Return dict matching `UserDetailResponse`.

- `get_user_activity(db, user_id, months=12)`:
  - Compute cutoff date: `datetime.now(timezone.utc) - timedelta(days=months * 30)`.
  - Query messages by month: `select(func.date_trunc('month', ChatMessage.created_at).label('month'), func.count(ChatMessage.id).label('count')).where(ChatMessage.user_id == user_id, ChatMessage.created_at >= cutoff).group_by('month').order_by('month')`.
  - Query sessions by month: same pattern with `ChatSession`.
  - Merge into list of `ActivityMonth` dicts. Use dict keyed by month string, then convert.
  - Return dict matching `UserActivityResponse`.

Import models: `User`, `UserCredit`, `File`, `ChatSession`, `ChatMessage`. Use lazy imports or direct imports as the codebase does.

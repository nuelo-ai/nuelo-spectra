   ```python
   user.last_login_at = datetime.now(timezone.utc)

   ```
   To:
   ```python
   user.last_login_at = datetime.now(timezone.utc)
   await db.commit()
   ```
   This fixes P29-T1 (last_login_at empty), P29-T2 (user detail last login), and P31-T2 (dashboard active_today = 0).

3. **backend/app/routers/admin/auth.py lines 43-47** -- Include minutes remaining in lockout error message:
   Change:
   ```python
   if now < locked_until:
       raise HTTPException(
           status_code=status.HTTP_429_TOO_MANY_REQUESTS,
           detail="Too many login attempts. Try again later.",
       )
   ```
   To:
   ```python
   if now < locked_until:
       minutes_remaining = max(1, int((locked_until - now) / 60) + 1)
       raise HTTPException(
           status_code=status.HTTP_429_TOO_MANY_REQUESTS,
           detail=f"Too many login attempts. Try again in {minutes_remaining} minute{'s' if minutes_remaining != 1 else ''}.",
       )
   ```
   This fixes P26-T6: lockout message lacks time information.

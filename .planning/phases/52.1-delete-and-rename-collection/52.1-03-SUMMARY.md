## Decisions Made

- Used `e.preventDefault()` on the DropdownMenuTrigger Button inside the Link — this blocks the Link's default navigation action without stopping event propagation entirely, which is the correct pattern for shadcn DropdownMenuTrigger
- The kebab menu on the detail page is rendered only when `!loadingCollection && collection` is truthy — this ensures the dialog never opens with a null collection prop
- `CollectionDetail` extends `CollectionListItem` (confirmed in workspace.ts types), so `collection` can be passed directly to dialog `collection` props without a type cast
- Used two separate `CreateCollectionDialog` instances in workspace/page.tsx (one for create, one for rename) to keep `open` and `collection` prop state fully isolated

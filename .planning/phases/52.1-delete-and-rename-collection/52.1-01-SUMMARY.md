## What Was Built

**CollectionService.delete_collection()** (`backend/app/services/collection.py`):
- Static async method: `(db: AsyncSession, collection_id: UUID, user_id: UUID) -> bool`
- Selects `Collection` filtering by `id` AND `user_id` — ownership enforced at query level
- Returns `False` if not found or not owned (caller receives 404)
- Calls `db.delete(collection)` + `db.flush()` then returns `True`
- DB cascade removes all child rows: signals, collection_files, pulse_runs, reports

**DELETE /{collection_id} endpoint** (`backend/app/routers/collections.py`):
- `@router.delete("/{collection_id}", status_code=status.HTTP_204_NO_CONTENT)`
- Parameters: `collection_id: UUID`, `current_user: WorkspaceUser`, `db: DbSession`
- Calls `CollectionService.delete_collection(db, collection_id, current_user.id)`
- Commits on success; raises `HTTP_404_NOT_FOUND` if service returns `False`
- No new imports needed — all required symbols already present

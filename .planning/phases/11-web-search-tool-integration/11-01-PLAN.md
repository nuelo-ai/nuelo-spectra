**1. Create `backend/app/services/search.py`** — SearchService class:
- Dataclasses: `SearchResult(title: str, link: str, position: int)` and `SearchResponse(query: str, results: list[SearchResult], success: bool, error: str | None = None)`
- `SearchService.__init__(self, api_key: str, num_results: int = 5, blocked_domains: list[str] | None = None, timeout: float = 10.0)`
- `SearchService.from_settings(cls) -> SearchService | None` — classmethod that reads from app settings. Returns None if `serper_api_key` is empty.
- `SearchService.search(self, query: str) -> SearchResponse` — async method using `httpx.AsyncClient` to POST to `https://google.serper.dev/search` with headers `{"X-API-KEY": self.api_key, "Content-Type": "application/json"}` and payload `{"q": query, "num": self.num_results}`.
- Parse `response.json()["organic"]` into SearchResult list, filtering out any result whose `link` contains a blocked domain.
- Handle `httpx.TimeoutException` (return success=False, error="timeout"), `httpx.HTTPStatusError` (return error=f"HTTP {status_code}"), and generic Exception.
- Log all searches using `logging.getLogger("spectra.search")` with structured JSON: `{"event": "web_search", "query": query, "results_count": N, "status": "success|timeout|error"}`.

**2. Create `backend/app/models/search_quota.py`** — SearchQuota model:
- Import Base from `app.models.base`.
- `SearchQuota(Base)` with `__tablename__ = "search_quotas"`:
  - `user_id: Mapped[UUID]` — ForeignKey to `users.id` with `ondelete="CASCADE"`, `primary_key=True`
  - `search_date: Mapped[date]` — `Date`, `primary_key=True` (composite PK)
  - `search_count: Mapped[int]` — `Integer`, default=0
- Use existing patterns from `user.py` and `file.py` models.

**3. Update `backend/app/models/__init__.py`** — add `from app.models.search_quota import SearchQuota` so Alembic can discover it.

**4. Update `backend/app/config.py`** — add search-related settings to Settings class:
- `serper_api_key: str = ""` — Serper.dev API key
- `search_max_per_query: int = 5` — max search tool calls per user query
- `search_daily_quota: int = 7` — daily search quota per user
- `search_num_results: int = 5` — results returned per Serper API call
- `search_timeout: float = 10.0` — Serper API timeout in seconds

**5. Create `backend/app/config/search.yaml`**:

  enabled: true

  num_results: 5
  daily_quota_per_user: 7
  timeout_seconds: 10.0
  blocked_domains: []

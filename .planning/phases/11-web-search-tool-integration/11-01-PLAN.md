**1. Create `backend/app/services/search.py`** — SearchService class:
- Dataclasses: `SearchResult(title: str, link: str, position: int)` and `SearchResponse(query: str, results: list[SearchResult], success: bool, error: str | None = None)`
- Class `SearchService` with `__init__(api_key: str, num_results: int = 5, blocked_domains: list[str] | None = None, timeout: float = 10.0)`
- Method `async def search(self, query: str) -> SearchResponse` — POST to `https://google.serper.dev/search` with `X-API-KEY` header, payload `{"q": query, "num": self.num_results}`, parse `organic[]` results, filter by blocked_domains, return SearchResponse. Handle `httpx.TimeoutException`, `httpx.HTTPStatusError`, and generic exceptions gracefully (return success=False with error message). Use `httpx.AsyncClient()` context manager per call.
- Method `async def search_multiple(self, queries: list[str]) -> list[SearchResponse]` — Use `asyncio.gather()` to execute queries concurrently with `asyncio.Semaphore(3)` as safety margin per RESEARCH.md.
- Logger: `logging.getLogger("spectra.search")` — log every search with structured dict: `{"event": "web_search", "query": query, "results_count": N, "status": "success|timeout|error"}` (SEARCH-07).
- Factory function `def get_search_service() -> SearchService | None` — loads from Settings (returns None if no API key or search disabled).

**2. Create `backend/app/models/search_quota.py`** — SearchQuota model:
- Import `Base` from `app.models.base`
- Class `SearchQuota(Base)` with `__tablename__ = "search_quotas"`
- Composite primary key: `user_id: Mapped[UUID] = mapped_column(ForeignKey("users.id", ondelete="CASCADE"), primary_key=True)` and `search_date: Mapped[date] = mapped_column(Date, primary_key=True)`
- Field: `search_count: Mapped[int] = mapped_column(Integer, default=0)`
- Class method `async def get_or_create(cls, db: AsyncSession, user_id: UUID) -> "SearchQuota"` — get today's record or create with count=0
- Class method `async def increment(cls, db: AsyncSession, user_id: UUID) -> int` — increment today's count, return new count
- Class method `async def get_remaining(cls, db: AsyncSession, user_id: UUID, daily_limit: int) -> int` — return daily_limit - today's count (min 0)

## Milestone v0.4 - Data Visualization

### Data Visualization Requirements

- Based on the query, user can ask the AI to generate the data visualization in the form of a chart
- Chart types include: Bar Chart, Line Chart, Pie Chart, and Scatter Plot
- If needed, the agent can generate 2-3 charts to show different perspectives of the data
- Subject to the data, the chart type is automatically selected unless specified by the user
- Data visuaization should be interactive  i.e., user should be able to hover over the chart to see the data/numbers/plot etc
- Data will be shown as a new section in the Data Card, nicely formatted and fit data card space
- Data visualization should be responsive
- User should be able to download the visualization as an image
